ComponentCreateFromXmlFile
function ComponentCreateFromXmlFile(const FileName: string; Owner: TComponent; const Name: string): TComponent;
Description
Create and read a component from the XML file with FileName. In order to successfully create the component from scratch, the component's class must be registered beforehand with a call to RegisterClass. Specify Owner to add the component as a child to Owner's component list. This is usually a form. Specify Name as the new component name for the created component.
